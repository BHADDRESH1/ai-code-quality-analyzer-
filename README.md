# Code Guardian Guard

Cleaned monorepo for local development and deployment.

## Structure

- `artifacts/code-analyzer`: frontend (Vite + React)
- `artifacts/api-server`: backend (Express)
- `lib`: shared libraries (`api-client-react`, `api-zod`, `db`)

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
